termos em negrito com a tag B

usando a tag <b>, porém contudo toda via entretanto a tag b esta para ser descontinuada, outro ponto a tag B não tem semantica, de preferencia para usar a tag <strong>;

termo en negrito com tag STRONG

usando a tag <strong> a frase tbm fica em negrito é um recurso atualizado tendendo a se manter por mais tempo, outro ponto muito interesante é que strong tem semantica.

termos em italíco com a tag I

a tag <i> tem o msm problem,a da tag <b> elas são antigas e não tem semântica.

termo em italigo usando a tag EM

a tag <EM> tem é parecida com o <strong> ela tem semantica e é mais recente

marcador de texto usando ata tag <MARK>

a tag <mark> é como se passase um marcado de texto porém a cor padrão dele e amarelo caso queira mudar é possivel de marcado em marcado com CSS porém se quiser trocar de todos pode fazer o CSS HEAD do html para criar um categoria de marcador.
