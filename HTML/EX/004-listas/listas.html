Tipos de listas

listas ordenadas.

listas ordenadas usam o comando <ol> dentro do OL é posiver colocar um expecificador type, também o start. O type troca como a listar vai ser ordenada por numeros, letras maiusculas, minusculas, e numeros romanos. ja o start é onde vai começar a lista. <li>para cada item da lista

  1. acordar
  2. levantar
  3. acender um cigarro
  4. ir ao banheiro
  5. tomar café
  6. ligar o computador
  7. estudar

listas não ordenadas

ja o comando <ul> é para listas não ordenadas o type dele é um pouco diferente podendo ser o padrão disk, circle e square

  • ir ao mercado
  • batata
  • cebola
  • beteraba
  • uva
  • miojo

lista de definições

a estrutura da lista de definições é criada com o comando <dl> já o foco da definições usam o comando <dt>e por fim a definição de fato usa o comando <dd>

beterraba
beterraba é uma planta herbácea da família das Quenopodiáceas, por Cronquist, ou das Amarantáceas, pela APG. Nome derivado do substantivo francês betterave. O colo tuberizado serve, para além dos fins culinários, produção de açúcar. Também existe uma variante cultivada para alimentação animal
