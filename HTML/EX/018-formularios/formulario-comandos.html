comandos do form

  1. action = para onde o formulario sera enviado
  2. method = como sera enviado o formulario (get e post)
  3. autocomplete = ligar o autocomplete (on e off)

comandos input

  1. required maxlength = a quantidade maxima de caracter
  2. required minlength = a quantidade minima de caracter
  3. required = faz o campo abrigatorio
  4. placeholder = deixa um aviso dentro do campo
  5. autocomplete = especifica o autocomplete(nome, nickname, tel)
